# Bili-Rss-YouGet

## 注意

此脚本为自用脚本，不保证其他机子能正常运行

## 功能

- [] telegram 通知，实时下载进度查看
- [] 最高清晰度下载（需要大会员）
- [] 邮件通知
- [] 封面图下载
- [] xml 转 ass
- [] 下载完成上传 OneDrive
- [ ] 一键脚本完成初始化设置
- [ ] 初次使用下载收藏夹以前的所有视频
